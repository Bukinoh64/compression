# Сжатие без потери качества

Веб-приложение для сжатия изображений и DOCX файлов без видимой потери качества. Работает полностью в браузере — никакой загрузки на сервер.

## Возможности

- **Сжатие изображений**: AVIF, WebP, PNG, JPG
- **Сжатие DOCX**: Оптимизация изображений внутри Word документов
- **Тёмная тема**: Удобная работа в любое время суток
- **100% конфиденциальность**: Файлы обрабатываются локально, не покидают ваш компьютер

## Как это работает

### Сжатие изображений

Приложение использует HTML5 Canvas API для оптимизации изображений:

1. **Масштабирование**: Если изображение больше 2000px по любой стороне, оно уменьшается с сохранением пропорций
2. **Квантование цветов**: Алгоритм уменьшает количество цветов до 256, группируя похожие оттенки
3. **Конвертация в WebP**: Для не-PNG/GIF изображений используется современный формат WebP с качеством 80%
4. **Сравнение**: Если сжатое изображение больше исходного — сохраняется оригинал

### Сжатие DOCX

DOCX — это ZIP-архив с XML и медиа-файлами внутри. Алгоритм:

1. **Распаковка**: DOCX распаковывается как ZIP-архив с помощью JSZip
2. **Поиск изображений**: Находятся все изображения (PNG, JPG, JPEG, GIF, WebP, BMP, TIFF)
3. **Оптимизация**: Каждое изображение масштабируется до 1200px и пересохраняется в оптимизированном формате
4. **Упаковка**: Файл запаковывается обратно с максимальным уровнем сжатия DEFLATE

## Технологии

- **HTML5** — Структура и Drag & Drop API
- **JavaScript (ES6+)** — Логика приложения
- **Canvas API** — Обработка и сжатие изображений
- **JSZip** — Работа с DOCX как с ZIP-архивом

## Использование

### Запуск

Просто откройте `index.html` в любом современном браузере:

```bash
# Windows
start index.html

# Или просто перетащите файл index.html в браузер
```

### Как использовать

1. Перетащите файлы в область загрузки или нажмите "Выбрать файлы"
2. Дождитесь завершения обработки
3. Файлы автоматически загрузятся на ваш компьютер
4. Просмотрите результаты сжатия (процент экономии)

## Результаты

Приложение показывает:
- Исходный размер файла
- Размер после сжатия
- Процент экономии

## Безопасность

- Все операции выполняются в браузере пользователя
- Файлы never не отправляются на сервер
- Никаких сторонних сервисов сжатия
- Полная конфиденциальность ваших данных

## Требования

- Современный браузер с поддержкой Canvas API (Chrome, Firefox, Edge, Safari)
- Для DOCX требуется подключение к интернету для загрузки JSZip (используется CDN)
